# 🏯ChatHistory
一个专注于历史模拟对话的智能体
## 🎯开发目标
1. 实现agent通过记忆机制进行交互
    - [x] 从指定文件中读取和构建记忆;
    - [x] 从记忆库进行检索记忆;
    - [ ] 构造一个聊天控制台;
    - [ ] 构造合适的prompt，测试选择合适的模型;
    - [ ] 协同LLM工作实现检索问答;
    - [ ] 从百科网页爬取历史人物介绍;
2. 实现多agent协作工作流
    - [ ] 实现不同角色信息的分离存储;
    - [ ] 实现多agent协同聊天机制;
3. 实现WebUI界面
